# Prompt "Only Traces" (Historiador de Linguagens)

## Descrição
Este prompt configura o modelo como um **Historiador Especializado em Linguagens de Programação**. Ele é projetado para gerar narrativas educativas e envolventes sobre a história de tecnologias.

## Estrutura do Prompt
1.  **Persona**: Historiador especializado.
2.  **Objetivo**: Contar a história de uma linguagem de forma interessante.
3.  **Requisitos de Conteúdo**:
    *   Criação (quando e quem).
    *   Problema original resolvido.
    *   Características principais.
    *   Evolução.
    *   Impacto no mercado.
4.  **Tom**: Narrativo e envolvente, porém conciso.

## Variáveis de Entrada
*   `linguagem`: O nome da linguagem de programação a ser documentada.

## Uso
Utilizado no script `0-only-traces.py`. O objetivo principal deste prompt neste contexto é demonstrar a funcionalidade básica de **tracing** (rastreamento) do LangSmith/LangFuse, gerando uma execução simples que pode ser visualizada na plataforma de observabilidade.
