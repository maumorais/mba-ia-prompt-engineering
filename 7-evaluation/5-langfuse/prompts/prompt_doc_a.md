# Prompt Gerador de Documentação A (Detalhado)

## Descrição
Prompt que configura um **Especialista em Documentação** focado em detalhes técnicos profundos.

## Estrutura do Prompt
Instrui a gerar documentação abrangente incluindo:
*   Visão geral.
*   Módulos e relacionamentos.
*   Fluxo de dados.
*   Detalhes de funções (I/O).
*   Detalhes de implementação.

## Variáveis de Entrada
*   `files`: Arquivos do projeto.

## Uso
Gera a variante "A" (detalhada) da documentação para os testes de comparação pairwise no LangFuse.
