# Prompt Gerador de Documentação B (Alto Nível)

## Descrição
Prompt que configura um **Engenheiro de IA** focado em documentação de alto nível e funcional.

## Estrutura do Prompt
Instrui explicitamente a:
*   Explicar o propósito e o "quê".
*   Ser factual e conciso.
*   **NÃO** incluir detalhes técnicos específicos (código, frameworks, implementações).

## Variáveis de Entrada
*   `files`: Arquivos do projeto.

## Uso
Gera a variante "B" (concisa) da documentação para os testes de comparação pairwise no LangFuse.
